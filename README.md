gobang
========
以 Qt5.7 为框架开发的基于基础博弈算法的人机五子棋博弈小游戏
-------------------------------------------------------

###运行环境
* win7，android

###核心算法
* 极大极小搜索
* alpha-beta 剪枝
* zobrist 哈希

###应用说明
* 程序采用无禁手游戏规则
* 玩家可选择先手或后手
* 玩家不能悔棋
